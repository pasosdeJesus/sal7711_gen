## Configuración 

* Configure el formato de la fecha en ```config/application.rb``` bien 
  con ```config.x.formato_fecha = 'yyyy-mm-dd'``` o  bien con 
  ```config.x.formato_fecha = 'dd-mm-yyyy'```
